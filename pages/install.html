--- title: "Installation instructions" toc: true --- h2#ppa-amd64 Using a PPA: Ubuntu 14.04 or newer (amd64) p. To install GLXOSD with multiarch support, simply copy and paste the following lines into the terminal: pre. sudo apt-add-repository -y ppa:nickguletskii200/glxosd \ && sudo apt-get update \ && sudo apt-get install -y glxosd glxosd-libs-amd64 glxosd-libs-i386:i386 p. To install GLXOSD without multiarch support, simply copy and paste the following lines into the terminal instead: pre. sudo apt-add-repository -y ppa:nickguletskii200/glxosd \ && sudo apt-get update \ && sudo apt-get install -y glxosd glxosd-libs-amd64 h2#ppa-i386 Using a PPA: Ubuntu 14.04 or newer (i386) p. To install GLXOSD, simply copy and paste the following lines into the terminal: pre. sudo apt-add-repository -y ppa:nickguletskii200/glxosd \ && sudo apt-get update \ && sudo apt-get install -y glxosd glxosd-libs-i386 h2#debian-amd64 Using the Debian repo: Debian jessie, stretch or sid (amd64) p. To install GLXOSD with multiarch support, simply copy and paste the following lines into a priviliged shell: pre. apt-get update \ && apt-get install -y software-properties-common apt-transport-https \ && apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 6F547B7B \ && apt-add-repository "deb https://glxosd.nickguletskii.com/repos/debian/ $(lsb_release -c -s) main" \ && apt-get update \ && apt-get install -y glxosd glxosd-libs-amd64 glxosd-libs-i386:i386 p. To install GLXOSD without multiarch support, simply copy and paste the following lines into a priviliged shell instead: pre. apt-get update \ && apt-get install -y software-properties-common apt-transport-https \ && apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 6F547B7B \ && apt-add-repository "deb https://glxosd.nickguletskii.com/repos/debian/ $(lsb_release -c -s) main" \ && apt-get update \ && apt-get install -y glxosd glxosd-libs-amd64 h2#debian-i386 Using the Debian repo: Debian jessie, stretch or sid (i386) p. To install GLXOSD, simply copy and paste the following lines into a priviliged shell: pre. apt-get update \ && apt-get install -y software-properties-common apt-transport-https \ && apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 6F547B7B \ && apt-add-repository "deb https://glxosd.nickguletskii.com/repos/debian/ $(lsb_release -c -s) main" \ && apt-get update \ && apt-get install -y glxosd glxosd-libs-i386 h2#debian-other Using the Debian repo: other Debian-based distributions p. To install GLXOSD on a Debian distribution that doesn't report itself as jessie, stretch or sid, you should follow the following general steps: ol li p. Install apt-transport-https, otherwise you won't be able to run apt-get update. li p. Import the public key by executing apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 6F547B7B. li p. Add deb https://glxosd.nickguletskii.com/repos/debian/ $RELEASE main to /etc/apt/sources.list, substituting $RELEASE for the closest Debian/Ubuntu release to your distribution (currently available: jessie, stretch, sid, trusty, vivid, wily, xenial, yakkety). li p. Update the APT package lists by executing apt-get update. li p. Install the GLXOSD packages by running apt-get install glxosd. If you have a multiarch system, you should also install glxosd-libs-i386 by running apt-get install glxosd-libs-i386. h2#other-distros Other distributions p. At the moment, there are no official packages for other distributions. If you would like to maintain GLXOSD packages for your favourite distribution, please consult the documentation. h2#source Compile from source p. Before compiling, please make sure the following libraries are installed and are available to CMake: FontConfig, Freetype, GLU, GLEW. p. You should also have git, CMake, and GCC installed. p. It is recommended that you use chroots to build GLXOSD. It is also recommended that you build both amd64 and i386 binaries because a lot of games are 32 bit applications. Since the instructions are distro-specific, the instructions for doing so can't be provided. pre. mkdir GLXOSD cd GLXOSD git clone -b v3 --recursive https://github.com/nickguletskii/GLXOSD source mkdir build cd build cmake -G "Unix Makefiles" ../source make all make install h2#disclaimer Disclaimer p.
