--- title: "Installation instructions" toc: true --- h2#ppa-amd64 Using a PPA: Ubuntu 15.10 or newer (amd64) p. To install GLXOSD, simply copy and paste the following lines into the terminal: pre. sudo apt-add-repository -y ppa:nickguletskii200/glxosd \ && sudo apt-get update \ && sudo apt-get install -y glxosd glxosd-libs-amd64 glxosd-libs-i386:i386 h2#ppa-i386 Using a PPA: Ubuntu 15.10 or newer (i386) p. To install GLXOSD, simply copy and paste the following lines into the terminal: pre. sudo apt-add-repository -y ppa:nickguletskii200/glxosd \ && sudo apt-get update \ && sudo apt-get install -y glxosd glxosd-libs-i386 h2#other-distros Other distributions p. At the moment, there are no official packages for other distributions. If you would like to maintain GLXOSD packages for your favourite distribution, please consult the documentation. h2#source Compile from source p. Before compiling, please make sure the following libraries are installed and are available to CMake: FontConfig, Freetype, GLU, GLEW. p. You should also have git, CMake, and GCC installed. p. It is recommended that you use chroots to build GLXOSD. It is also recommended that you build both amd64 and i386 binaries because a lot of games are 32 bit applications. Since the instructions are distro-specific, the instructions for doing so can't be provided. pre. mkdir GLXOSD cd GLXOSD git clone -b v3 --recursive https://github.com/nickguletskii/GLXOSD source mkdir build cd build cmake -G "Unix Makefiles" ../source make all make install h2#disclaimer Disclaimer p.
